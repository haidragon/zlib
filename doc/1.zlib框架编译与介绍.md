# zlib框架编译与介绍

## 一、zlib简介

zlib是一个通用的、无损的数据压缩库，使用DEFLATE算法进行数据压缩和解压。它由Jean-loup Gailly（压缩）和Mark Adler（解压）开发，是一个自由、开源的软件。

### 主要特性
- 无损数据压缩和解压
- 支持多种压缩级别
- 跨平台支持（Windows、Linux、macOS等）
- 内存占用小
- 速度快
- 支持流式处理

## 三、编译步骤

### 1. 配置编译选项
```bash
./configure
```

常用的configure选项：
- `--prefix=PATH`：指定安装路径
- `--help`：查看所有配置选项
- `--static`：编译静态库
- `--shared`：编译动态库（默认）

### 2. 编译
```bash
make
```

### 3. 安装（可选）
```bash
make install
```

### 4. 编译产物
编译成功后，会生成以下文件：
- `libz.a`：静态库文件
- `libz.so`（Linux）/ `libz.dylib`（macOS）：动态库文件

## 五、压缩级别

zlib支持9个压缩级别：
- `Z_NO_COMPRESSION` (0)：不压缩
- `Z_BEST_SPEED` (1)：最快速度
- `Z_BEST_COMPRESSION` (9)：最高压缩率
- `Z_DEFAULT_COMPRESSION` (-1)：默认压缩级别（通常为6）

## 六、错误码

常见的错误码：
- `Z_OK` (0)：成功
- `Z_STREAM_END` (1)：流结束
- `Z_ERRNO` (-1)：文件系统错误
- `Z_STREAM_ERROR` (-2)：流状态错误
- `Z_DATA_ERROR` (-3)：数据错误
- `Z_MEM_ERROR` (-4)：内存不足
- `Z_BUF_ERROR` (-5)：缓冲区错误

## 七、简单使用示例

### 压缩示例
```c
#include <zlib.h>
#include <stdio.h>
#include <string.h>

int main() {
    const char *source = "Hello, zlib!";
    uLong sourceLen = strlen(source) + 1;
    uLongf destLen = compressBound(sourceLen);
    Bytef *dest = malloc(destLen);
    
    if (compress(dest, &destLen, source, sourceLen) == Z_OK) {
        printf("压缩成功，原始大小: %lu, 压缩后大小: %lu\n", 
               sourceLen, destLen);
    }
    
    free(dest);
    return 0;
}
```

### 解压示例
```c
#include <zlib.h>
#include <stdio.h>
#include <stdlib.h>

int main() {
    // 假设compressed_data和compressed_len是已知的压缩数据
    Bytef *dest = malloc(original_size);
    uLongf destLen = original_size;
    
    if (uncompress(dest, &destLen, compressed_data, compressed_len) == Z_OK) {
        printf("解压成功: %s\n", dest);
    }
    
    free(dest);
    return 0;
}
```

## 八、流式处理示例

```c
#include <zlib.h>
#include <stdio.h>

void compress_stream(FILE *src, FILE *dest) {
    z_stream stream;
    unsigned char in[4096];
    unsigned char out[4096];
    int ret;
    
    // 初始化压缩流
    deflateInit(&stream, Z_DEFAULT_COMPRESSION);
    
    do {
        stream.avail_in = fread(in, 1, 4096, src);
        stream.next_in = in;
        
        do {
            stream.avail_out = 4096;
            stream.next_out = out;
            ret = deflate(&stream, stream.avail_in ? Z_NO_FLUSH : Z_FINISH);
            
            if (ret == Z_OK || ret == Z_STREAM_END) {
                fwrite(out, 1, 4096 - stream.avail_out, dest);
            }
        } while (stream.avail_out == 0);
    } while (ret != Z_STREAM_END);
    
    // 结束压缩
    deflateEnd(&stream);
}
```

## 九、注意事项

1. **缓冲区大小**：使用`compressBound()`计算压缩后的最大可能大小
2. **错误处理**：始终检查返回值，处理可能的错误
3. **内存管理**：确保正确释放分配的内存
4. **流式处理**：适合大文件处理，避免内存溢出
5. **版本兼容性**：使用`zlibVersion()`检查库版本
