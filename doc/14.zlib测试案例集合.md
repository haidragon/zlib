# zlib 测试案例集合

本目录包含20个zlib使用案例,涵盖各种API功能和应用场景。

## 测试案例列表

### 基础功能 (test2-test4)
- **test2**: 基础压缩/解压缩 - 使用compress/uncompress进行简单的压缩解压
- **test3**: 流式压缩 - 使用deflate进行流式压缩,适合大文件
- **test4**: 流式解压缩 - 使用inflate进行流式解压缩

### 压缩参数 (test5-test6)
- **test5**: 压缩级别对比 - 测试0-9级别的压缩率和速度
- **test6**: 大文件压缩 - 演示如何压缩/解压大文件

### 内存操作 (test7)
- **test7**: 内存压缩 - 在内存中进行压缩/解压缩,适用于网络传输

### 文件格式 (test8-test9)
- **test8**: Gzip文件操作 - 使用gzopen/gzread/gzwrite处理gzip文件
- **test9**: 原始数据压缩 - 压缩不带头部/尾部的原始数据

### 错误处理 (test10)
- **test10**: 错误处理 - 演示各种错误情况及其处理方法

### 高级特性 (test11-test13)
- **test11**: 进度回调 - 演示如何显示压缩/解压缩进度
- **test12**: 字典压缩 - 使用预设字典提高相似数据的压缩率
- **test13**: 部分解压缩 - 只解压缩数据的一部分

### 边界测试 (test14)
- **test14**: 压缩边界 - 测试各种边界情况下的压缩行为

### 校验和 (test15)
- **test15**: 校验和计算 - 演示Adler32和CRC32的使用

### 压缩策略 (test16)
- **test16**: 压缩策略 - 测试不同的压缩策略(Filtered/Huffman/RLE等)

### 自定义功能 (test17)
- **test17**: 自定义内存分配 - 设置自定义的内存分配函数

### 应用场景 (test18-test20)
- **test18**: 网络数据压缩 - 模拟网络传输的数据压缩
- **test19**: 图像数据压缩 - PNG格式图像数据压缩演示
- **test20**: 文本数据优化 - 文本数据的预处理和压缩优化

### 高级API (test21)
- **test21**: 高级特性 - deflateReset/deflateCopy/deflateParams等高级API

## 编译和运行

每个测试目录都包含独立的Makefile,会自动检测并编译zlib库(如果需要):

```bash
cd test/testXX
make          # 编译测试程序
make test     # 编译并运行测试
make clean    # 清理编译产物
```

## 目录结构

```
test/
├── test1/          # 原始测试案例
├── test2/          # 基础压缩
├── test3/          # 流式压缩
├── ...
├── test21/         # 高级特性
└── README.md       # 本文件
```

## 注意事项

1. 每个测试案例都可以独立运行
2. Makefile会自动检测zlib库是否存在,不存在会先编译zlib
3. 所有测试使用相同的zlib源码,确保一致性
4. 某些测试可能生成临时文件,运行后会自动清理
